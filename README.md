# Colorblind Resource Pack (Deuteranopia)

This is a WIP Minecraft resource pack designed to improve gameplay accessibility for players with deuteranopia, a form of red-green color blindness.

## What is Deuteranopia?

Deuteranopia is a type of color vision deficiency where the green color receptors (cones) in the eyes are missing or not functioning properly. This causes confusion between red and green colors, making them appear similar or indistinguishable. In Minecraft, this can make it difficult to distinguish between certain blocks, items, and UI elements that rely on red-green color differences.

## How This Pack Helps

This resource pack modifies textures to use alternative visual cues instead of relying solely on color differences. Common modifications include:

- **Block Textures**: Changing colors to enhance contrast on greenish/bluish blocks - WIP
- **Item Textures**: Enhancing contrast and adding distinguishing features - TODO
- **UI Elements**: Improving visibility of status indicators and menus - TODO
- **Particle Effects**: Making effects more distinguishable - TODO

The goal is to make the game more accessible while maintaining the visual appeal of Minecraft.

## Installation

1. Download the whole folder from GitHub
2. Launch Minecraft
3. Go to **Options** > **Resource Packs**
4. Click **Open Pack Folder** to locate your resource packs directory
5. Place the downloaded folder in the resource packs folder
6. Return to the Resource Packs menu and select the pack
7. Click **Done** and restart your world if necessary

## Compatibility

- **Minecraft Version**: 1.20-1.20.1 (Pack Format 15)
- **Multiplayer**: This resource pack is client-side only.
